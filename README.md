📊 Retail Sales Analytics (SQL)
🔹 Overview

This project analyzes retail sales data using SQL to generate insights on revenue, profit, customer behavior, and product performance.

🎯 Key Analysis Performed
Total revenue, profit, and order count
Top-selling products and profitable categories
Region-wise and monthly sales trends
Customer lifetime value (top customers)
Loss-making orders and low-profit sub-categories
Profit margin analysis


🛠️ SQL Concepts Used
Aggregations (SUM, AVG, COUNT)
GROUP BY, HAVING
Joins & Filtering
Window Functions (RANK, LAG, Running Total)
CTEs (Year-over-Year Growth)


📈 Key Insights
Identified top products driving revenue
Detected loss-making transactions
Analyzed sales trends over time
Evaluated regional performance
Found high-value customers


🚀 Conclusion

This project demonstrates strong SQL skills in data analysis, business insights generation, and advanced analytics using window functions.
